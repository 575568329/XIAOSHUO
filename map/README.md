## 二、快速修改

### 修改区域名称

打开 egions.json，找到对应区域：

```json
"北京市": { 
  "name": "王都·龙渊城",  // ← 修改这里
  "race": "human",
  "blood": "平原血脉",
  "status": "已开发",
  "danger": "低",
  "desc": "人类王国首都。"
}
```

### 修改种族

```json
"race": "human"  // 可选: human, orc, elf, dwarf, dragon, mermaid, abyss, goblin, undeveloped, none
```

### 修改状态

```json
"status": "已开发"  // 可选: 已开发, 部分开发, 未开发, 圣地, 禁地, 无人区
```

### 修改危险等级

```json
"danger": "低"  // 可选: 低, 低-中, 中, 中高, 高, 极高
```

## 三、添加新区域

在 egions 中添加：

```json
"新区域ID": {
  "name": "显示名称",
  "race": "种族",
  "blood": "血脉（可选）",
  "status": "状态",
  "danger": "危险等级",
  "desc": "描述"
}
```

## 六、常见操作

### 把一个区域从人类改为精灵

```json
"云南省": {
  "name": "南方密林",
  "race": "elf",        // ← 改这里
  "blood": "木精灵",    // ← 加血脉
  ...
}
```

### 把已开发改为未开发

```json
"status": "未开发",
"danger": "中"
```

### 修改主角起始位置

在 protagonist 部分：

```json
"protagonist": {
  "startLocation": "云南省",  // ← 改这里
  "startRegionName": "南方密林",
  "task": "丛林探险"
}
```
