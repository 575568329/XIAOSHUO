# 龙虾系统

一个程序员穿越到奇幻世界，带着「龙虾系统」看穿一切数据。

## 简介

主角在用OpenClaw开发游戏时穿越，复生到刚被哥布林灭团的佣兵小队。在他眼里，这个世界全是数据——血条、属性、概率。还有一只大龙虾帮他写战斗脚本。

不急着回家，先玩够了再说。
