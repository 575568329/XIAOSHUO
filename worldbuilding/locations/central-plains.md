# 中部平原 - 区域细化设定

> 更新于 2026-03-02

## 概述

中部平原是人类王国的核心区域，但并非完全开发。城市和主要道路周边相对安全，但偏远地区仍有未开发的危险地带。

## 佣兵系统设定

### 佣兵存在的原因

1. **未开发区域广泛** - 城市之间有危险地带
2. **魔兽生态** - 各种等级的魔兽威胁
3. **商业需求** - 商队护送、村庄保护
4. **探险需求** - 遗迹探索、宝藏猎人

### 佣兵任务类型

| 任务类型 | 描述 | 报酬 |
|----------|------|------|
| 清剿怪物 | 清除指定区域的怪物 | 中等 |
| 护送任务 | 保护商队/人员通过危险区 | 中等 |
| 狩猎任务 | 狩猎特定魔兽获取材料 | 高 |
| 探索任务 | 探索未知区域/遗迹 | 高（风险也高）|
| 守卫任务 | 保护村庄/设施 | 低-中等 |
| 调查任务 | 调查异常事件 | 中等 |

### 佣兵等级

| 等级 | 名称 | 能力范围 | 任务等级 |
|------|------|----------|----------|
| F级 | 学徒佣兵 | 只能打史莱姆、小动物 | F级任务 |
| E级 | 初级佣兵 | 能对付哥布林、狼群 | E-F级任务 |
| D级 | 中级佣兵 | 能单独对付小型魔兽 | D-E级任务 |
| C级 | 高级佣兵 | 能对付中型魔兽 | C-D级任务 |
| B级 | 精英佣兵 | 能对付大型魔兽 | B-C级任务 |
| A级 | 大师佣兵 | 能单独对付灾难级 | A-B级任务 |
| S级 | 传说佣兵 | 能对付传说级威胁 | 任意任务 |

## 主角起始设定

**起始位置：铁矿山区**

- 主角佣兵团接了一个清剿哥布林的任务
- 但这次哥布林异常强大（可能是裂缝魔气影响）
- 佣兵团被灭，主角复生到盗贼身上
- 幸存者 Garrick 带主角前往 **黑石关** 或 **月影镇**
