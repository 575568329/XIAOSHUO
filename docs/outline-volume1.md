## 🎬 主线剧情

### 第一阶段：穿越开局（第1-10章）

**核心事件：护送任务事故**

| 章节 | 内容 |
|------|------|
| 1-3 | 车祸穿越，附身16岁学徒，记忆碎片 |
| 4-6 | 带路人抄近道，遇中度魔兽，正主脑死亡 |
| 7-10 | 主角收拾残局，带领幸存者抵达铁壁要塞 |

**情感线**：困惑 → 恐惧 → 适应 → 求生

---

### 第二阶段：要塞生活（第11-25章）

**核心事件：融入要塞，拜师学艺**

| 章节 | 内容 |
|------|------|
| 11-15 | 交付物资，结识导师，成为学徒 |
| 16-20 | 学习斥候技能，系统初探索，挂机练级 |
| 21-25 | 接任务成长，结识要塞各色人物，日常 |

**情感线**：适应 → 成长 → 归属感

**关键人物**：
- 导师（师父）
- 要塞守备队长
- 其他学徒/冒险者
- 商人/铁匠等NPC

---

### 第三阶段：探索秘密（第26-40章）

**核心事件：发现裂缝痕迹**

| 章节 | 内容 |
|------|------|
| 26-30 | 接探索任务，深入幽暗密林 |
| 31-35 | 发现远古遗迹，裂缝痕迹，系统特殊解读 |
| 36-40 | 意识到裂缝被人为控制，返回报告 |

**秘密设定**：
- 裂缝痕迹 → 系统识别为"位面裂缝"
- 人为操控迹象 → 不明势力
- 伏笔：谁在控制？目的是什么？

**情感线**：好奇 → 惊讶 → 隐忧

---

### 第四阶段：魔兽暴动（第41-55章）

**核心事件：兽潮攻城**

| 章节 | 内容 |
|------|------|
| 41-45 | 兽潮爆发，要塞进入战备状态 |
| 46-50 | 守城战，主角参与战斗，导师掩护撤退 |
| 51-55 | 导师战死，主角悲痛自责，意识到能力不足 |

**导师之死**：
- 场景：掩护平民撤退
- 原因：兽潮规模超预期
- 主角心理：如果自己更强...

**情感线**：紧张 → 热血 → 悲痛 → 自责

---

### 第五阶段：告别出发（第56-70章）

**核心事件：突破瓶颈，离开要塞**

| 章节 | 内容 |
|------|------|
| 56-60 | 战后重建，埋葬导师，反思 |
| 61-65 | 化悲痛为动力，突破等级瓶颈（Lv.18-20）|
| 66-70 | 告别要塞朋友，决定前往龙石城学院 |

**核心动机**：
- 能力不足 → 师父战死
- 决心变强 → 保护朋友
- 查清真相 → 裂缝之谜

**情感线**：悲痛 → 决心 → 成长 → 期待
