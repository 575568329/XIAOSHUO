# 区域详细信息设计

## 信息字段

每个区域包含：
- **name**: 区域名称
- **race**: 种族
- **blood**: 血脉（可选）
- **status**: 状态（已开发/部分开发/未开发/圣地/禁地/无人区）
- **danger**: 危险等级（低/低-中/中/中高/高/极高）
- **resources**: 主要资源
- **monsters**: 主要魔兽
- **flight**: 是否禁飞（允许/限制/禁止）
- **terrain**: 地理环境
- **climate**: 气候
- **desc**: 简介
