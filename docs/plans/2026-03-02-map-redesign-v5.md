# 地图方案 v5（平衡版）

> 未开发占50%，各种族之间1-2个缓冲区

## 一、区域分配原则

### 比例
- 未开发：50%（约35个区域）
- 已开发：30%（约20个区域）
- 部分开发：20%（约15个区域）

### 缓冲区设计
- 各种族之间：1-2个未开发区域
- 不需要大量未开发，只需少量分隔
