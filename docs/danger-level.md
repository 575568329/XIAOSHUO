# 危险等级对照表

| 危险等级 | 适合等级 | 位阶 | 说明 |
|----------|----------|------|------|
| 低 | 1-15级 | 见习-初级初期 | 安全区，适合新手 |
| 低-中 | 10-20级 | 初级 | 边缘区，需谨慎 |
| 中 | 15-30级 | 初级-中级 | 危险区，组队推荐 |
| 中高 | 25-45级 | 中级-高级 | 高危区，精英才能生存 |
| 高 | 40-65级 | 高级-大师 | 极危区，死亡高发 |
| 极高 | 60级+ | 宗师+ | 禁地，九死一生 |
