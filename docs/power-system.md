## 🔀 职业系统

### 自由组合制

| 设定 | 说明 |
|------|------|
| 🎭 外表判断 | 看装备/风格推测职业 |
| 🔀 技能自由 | 啥都能学，无限制 |
| ⭐ 主职业 | 最擅长的，等级最高 |
| 📚 副职业 | 辅助技能，等级较低 |
| 🌟 天才标准 | 副职业≈主职业等级 |

**示例：**
- 普通人：剑士 Lv.35 + 炼金 Lv.12
- 天才：剑士 Lv.35 + 炼金 Lv.32

### 技能获取方式

| 方式 | 说明 |
|------|------|
| 💰 购买 | 技能书/卷轴，有钱就行 |
| ⛪ 神殿 | 找关系，信众优惠 |
| 🏫 学院 | 正规教育，学费贵 |
| 👨‍🏫 传授 | 高手收徒，看缘分 |
| 🧠 自创 | 融合创新（如：剑技+火魔法=烈焰斩） |

## 🔧 符文系统

### 分层设计

| 方式 | 说明 | 适用 |
|------|------|------|
| 📦 装备镶嵌 | 主要方式，安全 | 所有人 |
| 🩹 纹身 | 有副作用，需强韧体魄 | 兽人/龙裔等 |
| ⚙️ 魔导核心 | 法系增幅 | 法师/炼金师 |
| 🎴 符文卡片 | 仅日常魔法 | 便利工具 |

### 纹身规则

- 人类纹身 → 身体崩溃风险
- 兽人/龙裔 → 天赋体质，自带纹身
- 强行纹身 → 等级压制副作用

### 魔导核心

- **功能**：魔法增幅
- **制作者**：任何能使用魔法的人
- **使用方式**：装备镶嵌 OR 单核心裸奔
